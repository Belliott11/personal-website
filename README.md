Personal Website — Minimal Static Scaffold

Local preview

1. Open `index.html` in your browser (double-click) or run a simple server:

```
# Python 3
python -m http.server 8000
```

2. Visit http://localhost:8000

Deploy

- Push this folder to a GitHub repo and enable GitHub Pages from the `main` branch (or `gh-pages`).
